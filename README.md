# Patient Info

React JS Fundamentals Patient Info

## Features

- [x] React, ReactHooks
- [x] Material-UI, chart.js
- [x] 1. 환자 정보를 알 수 있는 테이블
- [x] 2. 필터 기능
- [x] 3. 테이블 클릭시 상세정보 보기 기능
- [x] 4. 테이블 컴포넌트 위 파이 그래프 추가
- [x] 5. 필터에 따라 차트값도 변경 가능
